使用說明 How to Use (V4)
  • Class list answer 可以完全留空 — 系統從頭自動排 Day / Time / Room
  • 在 Teacher Availability 填寫老師不可用時間（N = 不可用），其餘留空
  • 系統按優先次序排課：Mon → Tue → Thu → Wed → Fri
  • 同一班別（group）自動使用同一課室；如課室已滿自動轉用後備課室
  • 如 Class list answer 已填寫，系統沿用現有 Day/Time/Room（與 V3 相同）
