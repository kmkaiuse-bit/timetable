❌ 排不到老師？常見原因及解決方法

當系統顯示「未能分配 Lec1」，代表找不到符合條件的老師教這個班。原因通常是以下其中一項：

🗺️
H8 · 老師當天已在另一個 Centre 教書 例如：某老師早上在 CSW 教課，下午這班在 TW——系統不允許老師同一天去兩個 Centre。 解決方法：在下方時間表將這班拖移到另一天，令老師當天只需去一個 Centre。
⏰
H1 · 同一時段所有合資格老師都已有課 例如：星期二 0900 同時有 5 班數學，但只有 4 位數學老師——第 5 班沒有老師可教。 解決方法：拖移此班到另一個時段，或在 Teacher Load Table 加多一位老師。
📊
H4 · 老師每週 Loading 已達上限（6班） 每位老師每週最多教 6 班。如所有合資格老師均已達到上限，系統會放寬並發出警告。 解決方法：在 Teacher Load Table 增加老師或調高個別老師的班數上限。
📋
H7 · 沒有老師登記可以教這科 Teacher Load Table 中沒有任何老師登記教授此科目，或登記的老師 Quota 全為 0。 解決方法：在 Teacher Load Table 為此科目加入老師並填寫 Quota。
📖
S1 · 老師被排入申報不可用的時段（警告） 老師在 Teacher Availability 標記了 N，但系統找不到其他老師，只能安排他/她並發出警告。 解決方法：確認該老師是否真的不可用，或拖移班別到其他時段。

🔴 Hard Rules — 絕對不能違反

編號規則說明
H1 老師不能同一時段教兩班 同一時間同一位老師只能教一班
H2 課室同一時段只能用一班 座位亦不能超出課室上限
H3 學生上課地點須符合報名 Campus 報名 CSW 的學生，Core 科必須在 CSW 上課
H4 老師每週最多 6 班 1 loading = 1 班（無論 2小時或 4小時）
H5 CC Combine 須同科目 + 同語言 中文班與英文班不能 combine
H6 TKO Centre 特別時段 只可排 10:00–14:00 或 15:00–19:00（冷氣費問題）
H7 老師須具備教授資格 Teacher Load Table 有登記才能被分配
H8 老師同一天不能去兩個 Centre 例如早上 CSW、下午 TW = 不允許
H9 學生同一天不能去兩個 Centre 同一學生同日所有課須在同一 Centre

🟡 Soft Rules — 盡量符合，違反時系統發出警告

編號規則說明
S1 老師不可用時間偏好 Teacher Availability 填 N 的時段，系統優先跳過；萬不得已才排入並警告
S2 不要天地堂 同一半天（上午或下午）內，兩堂課之間不能有超過 2 小時空堂
S3 班別平均分佈到每天 同科目的班別輪流從 Mon→Tue→Thu→Wed→Fri 開始排，避免某天過份集中
