使用說明 How to Use
  • 上傳 Planning for Timetable.xlsx（包含已填好時間/課室的 Class list answer）
  • 如有老師變動，先更新 Teacher load table 再上傳
  • 如老師有不可用時間，在 Excel 加一張 Teacher Availability sheet：
    欄位格式：Teacher | Mon 0900 | Mon 1100 | … | Fri 1600
    填 N 表示該時段不可用，空白 = 可用
  • 系統自動分配 Lecturer 1（主教）及 Lecturer 2/3（後備）
  • 下載後如需調整個別老師安排，直接在 Excel 修改即可
